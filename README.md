# Interactive Resume

Frontend-only interactive resume built with React.

## Tech Stack

- React 19
- Create React App
- Tailwind CSS
- Framer Motion
- React Router DOM

## Run Locally

Use Node 20 or newer.

```powershell
cd frontend
npm install
npm start
```

Open:

```text
http://localhost:3000
```

## Build

```powershell
cd frontend
npm run build
```

The production files are generated in `frontend/build`.

## Resume Data

Edit the resume content in:

```text
src/data/resume.js
```

## Resume PDF

Place your PDF here:

```text
public/resume/Mohd-Anas-Shaikh-Resume.pdf
```

The Resume button downloads that file directly from the React public folder.
